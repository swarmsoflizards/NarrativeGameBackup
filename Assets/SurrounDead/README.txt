For the demo scenes to work properly please make sure you follow the steps below:
1. Set Rendering Path to Deferred: Edit-> Project Settings -> Graphics -> Set Rendering Path to Deferred
2. Set Rendering Color SpaceSet: Edit-> Project Settings -> Player -> Set Rendering Color Space To Gamma
3. Download and import Unity's Standard Assets Environment Package via Asset Store -> Standard Assets -> Environment (for water)
4. Download Post Processing Stack v2 from Window -> Packages -> Post Processing
5. Restart Unity to take effect.
